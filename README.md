# Página Web - Mantenimiento y Aseguramiento del Sistema Informático

## Descripción
Este proyecto consiste en una página web informativa sobre el mantenimiento y aseguramiento de la operación del sistema informático.

La página fue desarrollada utilizando:

- HTML
- CSS
- Visual Studio Code

## Contenido de la página

La página contiene:

- Menú de navegación
- Introducción
- Tipos de mantenimiento
- Aseguramiento de la operación
- Herramientas utilizadas
- Beneficios
- Conclusión
